汎用メッセージ制御プラグイン (fyis_MessageController)
Version: 1.2.0

■ 概要
RPGツクールMZのメッセージウィンドウに以下の機能を追加・拡張します。
- オートモード / スキップモード
- クイックセーブ / クイックロード
- バックログ機能
- ウィンドウレイアウトのカスタマイズ（背景画像、ボタン自動整列など）
- ボタン表示順のカスタマイズ（番号指定、0で非表示）
- 多段レイアウト（指定数ごとに折り返し配置）
- サブメニュートグル（メニューボタンで全ボタンを展開/収納）
- 名前ウィンドウの拡張（枠消去、位置調整、マスク無効化）
- メッセージウィンドウ背景画像の位置調整

■ 導入方法
1. 前提プラグイン `PluginCommonBase.js` を導入してください。
   （RPGツクールMZのインストールフォルダ -> dlc -> BasicResources -> plugins -> official などにあります）
   ※ Steam版の例: ...\steamapps\common\RPG Maker MZ\dlc\BasicResources\plugins\official\PluginCommonBase.js
2. 本プラグイン `fyis_MessageController.js` を `PluginCommonBase.js` より下に配置してください。
3. プラグイン管理画面で有効化し、必要に応じてパラメータを設定してください。
